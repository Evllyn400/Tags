O que são Tags?

São códigos de marcação de texto que possuem instruções, as tags são etiquetas, usadas para que o navegador exiba a página de acordo com a formatação definida pelos comandos HTML, elas são usadas para que você deixe palavras e textos do jeito que preferir, estilizando ela! Todas as tags tem um ínicio e um fechamento, e tudo que estiver entre os dois será o conteúdo dela. As tags podem tanto ser fechadas separadamente quanto funcionarem sozinhas Ex: "< i>Oi< /i>"< ou < input Oi/>. Existe também a possibilidade de usar duas tags juntas, por exemplo, você querer que um texto seja em itálico e também em negrito, para fazer isso só é preciso colocar uma ao lado da outra Ex: < i>< strong>Tudo bem?< /strong>< /i>.

Seguindo nessas diversas regras das tags, há também em algumas delas a necessidade de serem usadas as tags filhas, que são uma tag dentro da outra, "ol" e "ul" por exemplo precisam de uma tag filha para que possa ser colocado os itens desta lista, e para que funcione da maneira correta é preciso repetir o comando do atributo a cada item da lista.

O que são Atributos?

São propriedades que uma tag pode ter, palavras especiais para controlar o comportamento do elemento, cada atributo deve ser escrito dentro de uma tag para que possa funcionar Ex: < strong style = "color: pink;">Amor< /strong> Resultado: Amor

Tipos de atributos

Existem dois tipos de atributos, os Globais e Específicos, os atributos globais são genéricos, usados em quase tudo, já os específicos são atributos que só podem ser usados atuando em uma tag específica

Algumas tags e suas expecificações

  1. "title" = Adiciona um título
  2. "i" = Deixa o texto selecionado em itálico
  3. "strong" = Transforma em negrito
  4. "u" = Traça uma linha, sublinhando
  5. "mark" = Colore a frase como um marca texto
  6. "input" = Abre um campo de texto
  7. "img" = Adiciona uma imagem selecionada na página
  8. "h1-h6" = É uma TAG como a "title" porém com tamanho, e esses tamanhos vão do h1 ao h6, adicionando assim, subtítulos
  9. "p" = Da ínicio a um parágrafo, textos
  10. "blockquote" = Para fazer citações ou observações
  11. "sup" = Elevado à Ex: Diome
  12. "ol" = Lista ordenada, usa marcação com números
  13. "ul" = Lista não ordenada, usa marcação com pontos
  14. "a" = Para adicionar um link
  15. "hr" = Traça uma linha, separando a página
  16. "small" = Deixa o texto pequeno
  17. "font" = Serve para escolher tanto a cor da fonte quanto seu tamanho*
  18. "del" = Representa algo que foi excluído, risca a palavra*
  19. "abbr" = Indica uma abreviação ou uma sigla*

Atributos das Tags

    Globais:

  1. "id" = Identifica o elemento que você escrever na frente para que possa modificá-lo
  2. "style" = Coloca o CSS direto no elemento selecionado
  3. "class" = Padroniza o que você quer no estilo que escolher

    4. Específicas:

  4. "type" = Específico da TAG "input", diz o conteúdo usado na TAG Ex: "number" ou "text"
  5. "src" = Específico da TAG "img" trás um endereço de imagem para que ela possa aparecer
  6. "width" = Para definir a largura da imagem
  7. "href" = Adiciona o link da TAG "a"
  8. "target" = Usado na TAG "a" para abrir o conteúdo do link em uma nova aba (usando a função "_blank")
